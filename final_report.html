Abstract

Modern portfolio theory is a concept from finance to describe ways of diversifying and allocating assets in a finacial portfolio in order to maximize the porfolios expected return given some level of risk tolerance. Using modern portfolio theory an investor bundels different types of investments together, so that when some of the securities fall in value other rise. So in that case the return of each individual asset is less important then the behaviour of the entire portfolio. American economist Harry Markowitz first introduced Modern portfolio theory in 1952 what led to him being awarded the Nobel Prize in Economics in 1990. Since that time many researchers have developed theories on portfolio optimization and risk management. Aim of this work is to implement some of the approaches and to compare them.

Monte Carlo simulation

Monte Carlo Simulation is a mathematical technique that generates random variables for modelling risk or uncertainty of a certain system. Monte Carlo simulation is often used in finance, engineering and physics. Monte Carlo simulation produces distributions of possible outcome values. This is the easiest way of portfolio optimization, so we will start with it. In our case we use uniform distribution. All values have equal chanses to occur. We define minimum and maximum as 0 and 1.

We make our simulations 2000 times. Each time different weights(that sum to one) are generated. Based on those weights return, volatility and sharpe ratio of portfolio is calculated. Sharpe ratio is a measure of risk-adjusted return. We store all the combinations of weights.

num_assets <- length(tickers)
num_portfolios <- 2000

portfolios <- data.frame(matrix(NA, nrow=num_portfolios, ncol=7))

for (port in 1:num_portfolios) {
  weights <- runif(n = num_assets, min = 0, max = 1)
  weights <- weights/sum(weights)
  returns <- dot(weights, returns_annualy)
  volatility <- sqrt(dot(t(weights), dot(cov_annualy, weights)))
  sharpe = returns/volatility
  portfolios[port,] <- c(port, returns, volatility, sharpe, weights)
  }

# Calculate portfolio returns
portfolio_returns = function(x) {
  port.returns = 0
  
  # Multiplication of the i-th asset by the i-th weight in "x"
  for (i in 1:length(x)) {
    port.returns = port.returns + asset_returns[,i] * x[i]
  }
  
  return (port.returns)
}
# Using Sharpe ratio as an objective function
sharpe = function(x) {
  port.returns = portfolio_returns(x)
  
  return (mean(port.returns)/sqrt(var(port.returns)))
}
# Penalty function keeps track of basic constraints for weights
penalty = function(x) {
 fine = 100 * abs((sum(x)-1))   # "sum x = 1" constraint
 
 for (i in 1:length(x)) {
  fine = fine + 
     10 * max(c(0,x[i]-1))**2 +  # "x <= 1" constraint
     10 * max(c(0,-x[i]))**2     # "x >= 0" constraint
 }
 
 fine
}
# Function to be optimized
obj = function(x) {
  # We want the maximum Sharpe ratio, so we multiply it by
  # -1 to fit an optimization problem
   sharpe(x) - 100*penalty(x)
}
